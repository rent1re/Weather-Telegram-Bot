# 🌤 Weather Bot

Телеграм-бот для получения информации о погоде в любом городе мира.

## Возможности

- Текущая погода в любом городе
- Прогноз на 7 дней
- Погода на завтра
- Погода за вчера

## Технологии

- Python 3.10+
- pyTelegramBotAPI
- WeatherAPI

## Установка

1. Клонируй репозиторий:
```
git clone https://github.com/YOUR_USERNAME/weather-bot.git
cd weather-bot
```

2. Установить зависимости:
```
pip install -r requirements.txt
```
или же аналог если у вас не получилось - '''
py -m pip install -r requirements.txt
'''
3. Запусти бота:
python bot.py
